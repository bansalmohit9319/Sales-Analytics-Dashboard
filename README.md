# Sales Analytics Dashboard

An interactive Sales Data Analytics Dashboard built using Streamlit, Pandas, and Plotly for analyzing sales performance, profit trends, regional sales distribution, and category-wise business insights.

The dashboard allows users to upload custom sales datasets, apply dynamic filters, and visualize key business metrics through interactive charts and analytics components.

---

## Features

- Upload custom CSV sales datasets
- Interactive sidebar filters
- KPI metrics dashboard
- Sales analysis by region
- Category-wise sales visualization
- Monthly sales trend analysis
- Top selling products analysis
- Profit vs Sales analysis
- Download filtered datasets
- Interactive Plotly visualizations

---

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly

---

## Dashboard Functionalities

### KPI Metrics
- Total Sales
- Total Profit
- Total Orders
- Average Sales

### Interactive Analytics
- Region-wise sales analysis
- Category-wise sales distribution
- Monthly sales trends
- Top selling products
- Profit analysis dashboard

### Dataset Features
- CSV file upload
- Data cleaning
- Duplicate handling
- Missing value handling
- Export filtered dataset
